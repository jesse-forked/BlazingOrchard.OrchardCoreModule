# BlazingOrchard.Site

BlazingOrchard.Site is the standard Orchard Core site theme included with BlazingOrchard.

## Versioning

This theme follows the BlazingOrchard five-part compatibility version:

```text
{orchard-major}.{orchard-minor}.{orchard-patch}.{blazing-security}.{blazing-bug}
```

Current compatibility version: `3.0.0.0.0`.

- The first three parts match the Orchard Core version this theme has been tested against.
- The fourth part is the BlazingOrchard security-fix counter.
- The fifth part is the BlazingOrchard bug-fix counter.

Example progression:

- `3.0.0.0.0` initial release for Orchard Core `3.0.0`.
- `3.0.0.0.1` bug fix.
- `3.0.0.1.2` security fix plus bug fixes.
- `3.0.1.2.1` compatibility update after testing against Orchard Core `3.0.1`.
